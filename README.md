# 扫雷游戏 - 现代版 ⚡

一款视觉风格现代的经典扫雷游戏实现，采用时尚深色主题和多种游戏模式。

## 🌟 游戏特色

- **经典与倒计时模式** ⏱️：玩传统计时版本，或挑战倒计时模式
- **多种难度级别** 🎚️：从初级、中级、专家到自定义棋盘大小
- **时尚深色主题** 🌙：护眼深色设计，搭配渐变背景和细腻动画
- **响应式布局** 📱💻：自适应不同屏幕尺寸，优化单元格大小
- **触控与鼠标双支持** 🖱️👆：直观控制，兼顾移动设备和桌面端
  - 点击/轻触揭示单元格
  - 右键/长按标记地雷
  - 双击数字可在标记数量匹配时清除周围单元格
- **视觉反馈** ✨：动画星星背景、悬停效果和状态指示器

## 🎮 玩法说明

1. 选择您偏好的难度（棋盘大小和地雷数量）
2. 选择游戏模式（经典计时或倒计时挑战）
3. 倒计时模式下，选择时间限制
4. 点击"开始游戏"按钮
5. 揭示单元格，找出所有非地雷方块，避免触雷
6. 标记疑似地雷，防止误点
7. 清除所有非地雷方块即可获胜！

## 🕹️ 操作控制

- **左键点击/轻触**：揭示单元格
- **右键点击/长按**：放置/移除标记
- **双击**：当标记数量与数字匹配时，清除周围单元格

## ⚙️ 自定义选项

- **棋盘尺寸**：
  - 初级：9×9，10个地雷
  - 中级：16×16，40个地雷
  - 专家：30×16，99个地雷
  - 支持自定义尺寸

- **游戏模式**：
  - 经典模式：计时游戏，无时间限制
  - 倒计时模式：需在时间结束前完成游戏

## 🚀 安装说明

1. 克隆本仓库
2. 在任何现代浏览器中打开 `扫雷-模式选择.html`
3. 无需额外依赖！

## 💻 技术细节

- 使用原生HTML、CSS和JavaScript构建
- 采用CSS变量实现一致的主题风格
- 响应式设计，配合媒体查询
- 基于Canvas的星星动画背景
- 支持触控的事件处理
